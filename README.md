# fullstack-auth-system
A full-stack authentication system built with Next.js 14, MongoDB, and JWT. Features include user registration, email verification, login/logout, forgot password, and password reset — all with secure HTTP-only cookies and protected routes.

A production-ready authentication system built with Next.js App Router, 
MongoDB (Mongoose), and JWT. Implements the full auth lifecycle: sign up, 
email verification via tokenized links, secure login with HTTP-only cookies, 
and a complete forgot/reset password flow using time-limited tokens and Nodemailer.
